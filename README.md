# Projeto Daily Study
O Daily Study é um projeto para compartilharmos nosso dia a dia de estudos. Ele atualmente conta com um feed interativo e cronológico, aba de perfil customizável, com fotos de perfil, banner e biografia. Por enquanto, contamos com apenas 4 telas:

-Login
-Registro
-Feed
-Perfil

Nosso objetivo com o projeto social seria postarmos tudo o que estudamos e fazer um breve resumo, para que possamos encorajar uns aos outros a ter mais gosto pelos estudos, nem que seja pela competição. Ele está passando por alterações, pois nem sempre temos tempo livre para mexer nele; no momento, estamos desenvolvendo com ajuda de IA, pois nosso conhecimento não é dos melhores, mas o principal foco deste projeto é aprender um pouco mais sobre linguagens e banco de dados.
